1. Основное свойство display: grid включает CSS Grid. В отличие от одномерного Flexbox, Grid - двумерная система, позволяющая одновременно управлять строками и столбцами, создавая сложные макеты с точным позиционированием.

2. grid-template-columns определяет явные столбцы по шаблону, а grid-auto-columns задает размер для неявно создаваемых столбцов. Первое используется для известной структуры, второе - для динамически добавляемых элементов.

3. repeat() повторяет шаблон (например, repeat(3, 1fr)), minmax() задает диапазон размеров (minmax(100px, 1fr)), auto-fit автоматически размещает столбцы, заполняя доступное пространство.

4. Именованные области через grid-template-areas позволяют визуально описывать макет. Преимущества: наглядность, легкая перестановка элементов без изменения CSS, упрощение адаптивности.

5. Grid-линии нумеруются от 1 до n+1. grid-column: 1/3 размещает элемент от 1-й до 3-й линии. Можно использовать отрицательные числа для отсчета с конца.

6. gap создает равномерные промежутки только между элементами сетки, не влияя на внешние отступы. Это устраняет проблемы с margins типа схлопывания и обеспечивает предсказуемость.

7. justify-items выравнивает содержимое ячеек по горизонтали, align-items - по вертикали. Они работают внутри каждой ячейки сетки, в отличие от justify-content/align-content, которые выравнивают саму сетку.

8. auto-fill создает максимальное количество столбцов (включая пустые), auto-fit растягивает существующие столбцы на все пространство. Разница видна когда элементов меньше, чем может поместиться.

9. Создание макета: grid-template-areas: "header header" "sidebar content" "footer footer". Правила: имена в кавычках, пробелы разделяют ячейки, точка для пустой ячейки, одинаковые имена создают области.

10. Преимущества Grid: точное двумерное позиционирование, контроль над строками и столбцами одновременно, минимальный код для сложных макетов, мощные функции авторазмещения, простое создание адаптивных сеток без медиа-запросов.
