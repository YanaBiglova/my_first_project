1. Основное свойство - display: flex. Дочерние элементы становятся flex-элементами и выстраиваются вдоль основной оси.

2. justify-content - выравнивание по основной оси (например, space-between)

align-items - выравнивание по поперечной оси (например, center)

3. flex-direction задает направление основной оси. Значения: row, row-reverse, column, column-reverse. Определяет, как элементы располагаются в контейнере.

4. flex: 1 =

flex-grow: 1 (может увеличиваться)

flex-shrink: 1 (может уменьшаться)

flex-basis: 0 (базовый размер)

5. flex-wrap позволяет переносить элементы на новую строку. Используется при нехватке места.

6. align-items - для одной строки

align-content - для нескольких строк (при flex-wrap: wrap)

7. order меняет порядок элементов. Полезно для адаптации контента без изменения HTML-структуры.

8. Основная ось - направление расположения элементов (зависит от flex-direction). Поперечная - перпендикулярна ей.

9.
.container {
  display: flex;
  flex-wrap: wrap;
}
.item {
  flex: 1 1 300px; /* Растягивается, сжимается, минимум 300px */
}
10. gap проще в использовании, создает равные промежутки только между элементами (не по краям), не требует сложных селекторов.
